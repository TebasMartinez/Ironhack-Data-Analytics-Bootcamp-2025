## Notes

In a matrix, the columns are referred to as:
- variables,
- features, or
- artibutes,
while rows are normally referred to as:
- observations,
- data points,
- records,
- samples, or
- examples.

Types of arrays based on their dimensions:

| Dimensions | Name |
| -------- | ------- |
| 0 | Scale |
| 1 | Vector |
| 2 | Matrix |
| 3 | Tensor |
